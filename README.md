# react-native-example
A example of React Native

# Requisites
- NodeJS
- Expo CLI (`npm install -g expo-cli`)
# Steps
1. Run `expo init FirstReactNativeApp`
3. `cd FirstReactNativeApp`
4. Run `npm start`
